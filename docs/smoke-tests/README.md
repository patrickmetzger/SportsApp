# Smoke Tests

Manual walkthrough scripts for each user role. Each script covers the happy path and a few edge cases. No technical knowledge required — just follow the steps and note any issues.

## Test Accounts

Ask PJ for the test credentials before starting. You will need one account per role you are testing.

| Role | What you test |
|------|---------------|
| [Admin](./admin.md) | Platform-wide management: schools, users, programs, certifications |
| [School Admin](./school-admin.md) | School-level management: coaches, programs, students, comms |
| [Coach](./coach.md) | Program submission, certification tracking, attendance |
| [Assistant](./assistant.md) | Certification upload, attendance |
| [Parent](./parent.md) | Registration, communications |

## How to Report Issues

Note the role, the page URL, the step number, and what you expected vs. what happened.
